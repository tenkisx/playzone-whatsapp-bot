## Como funciona uma encomenda por WhatsApp
1. O cliente diz o que quer (ex.: "quero um Samsung S23").
2. O assistente procura o produto real (preço/stock) via WooCommerce.
3. Confirma modelo, quantidade e preço com o cliente.
4. Recolhe: nome, telefone, morada/cidade de entrega.
5. Cria a encomenda no sistema e envia o número + link de pagamento.
6. A equipa é alertada para dar seguimento.

## Quando escalar para humano (não tentar resolver sozinho)

- Pedidos fora do catálogo ou empresariais (grandes quantidades).
